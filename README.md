# Лабораторная работа ООП на С#

## Автор
Гинжул Мария

## Цель
Освоить принципы объектно-ориентированного программирования: наследование, полиморфизм, инкапсуляцию, а также изучить применение паттерна проектирования Singleton на примере создания иерархии классов животных.

## Описание
Программа реализует иерархию классов животных. Базовый класс Animal содержит общие характеристики, а классы-наследники добавляют уникальные свойства. Управление коллекцией животных осуществляется через класс  AnimalManager, который предоставляет консольное меню для взаимодействия с пользователем.
